# Module 1: Fundamentals

Get started with AI agents — understand the landscape, build your first agent, and survey the major frameworks.

## Prerequisites

- Python 3.12+
- `uv` package manager
- NVIDIA API Key and Tavily API Key set in the root `.env` file

## Notebooks

| Notebook | Description |
|----------|-------------|
| `01_Introduction.ipynb` | Course overview, module structure, and prerequisites |
| `02_Build_Your_First_Agent.ipynb` | Build a RAG agent from scratch using LangGraph, Tavily, and FAISS |
| `03_Surge_of_Agents.ipynb` | Survey of agent frameworks: OpenAI, LangChain, LangGraph, and CrewAI |
